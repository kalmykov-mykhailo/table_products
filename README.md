# Таблиця товарів

## Технології

- [Vite](https://vitejs.dev/)
- [Vue 3](https://v3.vuejs.org/)

## Встановлення

```bash
# Клонувати репозиторій
git clone https://github.com/kalmykov-mykhailo/table_products.git

# Перейти в каталог проекту
cd table_products

# Встановити залежності
npm install
# або
yarn install

# Режим розробки
npm run dev
# або
yarn dev

# Збірка для продакшн
npm run build
# або
yarn build
